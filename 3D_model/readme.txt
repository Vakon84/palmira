3D модели корпуса. Основная часть, клавиатура и верхняя крышка. Файлы открываются в Openscad.
Содержание файлов:
enter.stl - крышка для кнопки ввод
probel.stl - крышка для кнопок пробел
kb_palmira.stl - верхняя крышка для клавиатуры
verh_palm.stl - верхняя крышка корпуса
корпус_пальмира.stl - основной корпус
